# SQL Data Analysis - DecodeLabs Project 3

## 📌 Project Overview

This project is part of the **DecodeLabs Data Analytics Industrial Training Program (2026)**. The objective is to analyze the Superstore dataset using SQL and extract meaningful business insights through various SQL queries.

---

## 🎯 Objective

* Perform SQL-based data analysis on the Superstore dataset.
* Practice SQL fundamentals such as data retrieval, filtering, sorting, grouping, and aggregation.
* Generate business insights from sales and profit data.

---

## 🛠️ Tools Used

* MySQL Workbench
* SQL
* Superstore Dataset (CSV)

---

## 📂 Dataset

**Dataset Name:** Superstore.csv

The dataset contains information about:

* Orders
* Customers
* Products
* Categories
* Sales
* Profit
* Discounts
* Regions
* States

---

## 📚 SQL Concepts Covered

* SELECT
* WHERE
* ORDER BY
* GROUP BY
* HAVING
* LIMIT
* COUNT()
* SUM()
* AVG()
* MIN()
* MAX()

---

## 📊 Analysis Performed

* Displayed complete dataset
* Retrieved selected columns
* Filtered records using WHERE
* Sorted data using ORDER BY
* Calculated total sales and profit
* Found average sales
* Counted total orders
* Grouped data by Category, Region, State, and Segment
* Identified top-selling products
* Identified top customers
* Analyzed profit by state and sub-category
